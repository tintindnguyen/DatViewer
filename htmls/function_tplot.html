Syntax

tplot(panel_id, source_id, plot_variable,line_id)
line_id input is optional

Plotting Variable

The plotting variable must meet 3 requirements
  1. A struct with 2 field names: name and value
  2. The name field must be the same as the variable name
  3. The value field must be the same size as the source's time vector
The variable's value field can be empty. When the varaible is not loaded and exists in the source' data, tplot validates the data and loads in before plotting.
