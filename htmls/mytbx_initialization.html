Launch GUI

d.launch_gui

Import Data

How To

Import new souce file to the tool. The tool can store up to 4 sources at a time. Without declaring the source's location, DatViewer uses the first available slot. The example below load 2 sources into slot 1 and slot 3
d.import_new_source("tabledata1.txt");
d.import_new_source("tabledata2.txt",3);
When import_new_source is called again with a slot that is already taken, DatViewer overides the old source
d.import_new_source("tabledata2.txt",1);

Data type Format

ASCII Data Format

The tool assumes the ASCII data contains list of variable names on the 2nd line seperated by '\t', ' ', ','. The first data column is 'time'.

MAT Data Format

Mat file is assumed to contain one variable with 'table' data type format.
