Create Panel

d.create_panel(5);

Plot Time Data

tplot function plots a specified source's data onto a specified panel number. The data list exists in the source's data struct
d.tplot(6,2,var1); % Plot variable 'var1' that is relevant to source 2 onto panel 6
Each panel contains maximum of 6 lines. tplot function plots the data in sequential order from line 1 to line 6. When a panel already containing 6 lines, tplot takes an additional argument to select a replacement line in the panel. tplot accepts line number input when the panel is empty.
d.tplot(6,2,var1,4); % Replace the 4th line in panel 6 with var1 that is relevant to source 2
